# maintenance_management

A beginner-friendly Frappe custom app scaffold for tracking maintenance breakdown tickets.

## What is included

- Basic Frappe app structure
- One custom DocType: `Maintenance Breakdown Ticket`
- Priority SLA helper logic
- Simple client-side status guidance

## Planned workflow states

- Open
- Inspection Pending
- Indent Pending
- Spare Check Pending
- Rectification Pending
- Resolution Confirmation Pending
- Management Approval Pending
- Closed

## Priority SLA mapping

- Red: 24 hours
- Yellow: 48 hours
- Green: 72 hours
